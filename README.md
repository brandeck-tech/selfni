# سلفني - نظام ادارة الديون الذكي

## تشغيل المشروع
bash start.sh

## التقنيات
- Backend: Node.js + TypeScript + Express + PostgreSQL
- Frontend: React + TypeScript + Vite

## API
- POST /api/auth/login
- GET  /api/debts
- GET  /api/customers
- GET  /api/risk/clients
- GET  /api/installments/debt/:id
